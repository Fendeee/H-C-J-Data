什么是CSS的元素显示模式

元素（标签）以什么方式进行显示，比如：div独占一行，sapn一行可以放多个。

HTML一般分为《块元素（div）》和《行内元素（span）》

《块元素》

常见的块元素有：h1-h6、p、div、ul、ol、li等。div是最典型的

1..独占一行

高度，宽度，外边距和内边距等都可以控制。

《行内元素》

常见元素有，a、strong、b、em、i、del、ins、u、span等。

span是最典型的行内元素

1.一行可以放多个元素，2.高、宽直接设置是无效的， 3.默认宽度是它本身的宽度，（有多少字符占多宽长度）4.行内元素只能容纳文本或者其他《行内元素》不允许之外的标签元素。

链接里面可以转换块级模式然后放《行内元素》但不能再放其他链接

《行内块元素》

行内块元素中的特殊标签：img（图片）、input（表单）、td（表格）他们都具有《块元素》和《行内元素》的特点

和相邻行内元素（内元素）在一行上，但是之间有空白间隙，一行可以显示多个（行内元素特点）
默认宽度是它本身的宽度（行内元素特点）
高度、行高、外边距和内边距都可以控制（块元素特点）

元素总结

元素模式元素排列设置样式默认宽度包含
块级元素一行内只能放一个块级元素可设置宽、高容器的100%容器级可包含任何标签
行内元素一行可放多个行内元素不可直接设置宽、高它本身内容的宽度容纳文本或其他行内元素
行内块元素一行放多个行内块元素可设置宽、高它本身内容的宽度
