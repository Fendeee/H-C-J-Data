CSS3中可以通过box-sizing来指定盒模型,有2个值:即可指定为content-box、border-box
这样我们计算盒子大小的方式就发生了改变。

box-sizing: content-box;盒子大小为width + padding + border ( 以前默认的)
box-sizing: border-box盒子大小为width
