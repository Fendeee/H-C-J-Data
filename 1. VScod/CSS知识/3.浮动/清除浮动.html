清除浮动--1.额外标签法

下浮动盒子后添加新的盒子隔开，.类名{clear: both;}清楚浮动，或者<div style="clear:both"> < div >br
但是必须是块元素，不能是行内元素

父级添加 overflow

给父级元素添加overflow属性，将其属性值设置为hidden/auto/scroll

after伪元素

语法：.类名::after {
content:''（必须属性）;
display:block;（转换成块元素）
height: 0;
clear:both;（清除）
visibility:hidden;(隐藏，不看见此元素)
}
.类名 *zoom:1;（IE6，IE7可以来清楚浮动）

双伪元素清除浮动

.类名::before .类名::after {
content: '';
display: table;(转换为块级元素并且一行显示)
}
.类名 ::after {
clear: both;(清楚浮动)
}
