转换(transform)是CSS3中具有颠覆性的特征之一，可实现元素的位移，旋转，缩放等效果，
转换(transform)可以理解为变形

transform(移动)
transform: translate(x,y); (可分开写)
transform: translateX(n);
transform: translateY(n)
transform: translate(50%,50%) 水平居中
transform: translate(-50%, -50%); 水平居中
搭配position: relative;和absolute来用
定义2D转换中的移动，沿着x，y轴移动元素
transform不会影响其他元素的位置，transform中的百分比单位
是相对自身元素的transform:(50%,50%);
对行内元素标签没有效果
rotate(旋转)
transform: rotate(度数deg);
正角度则为顺时针，负时则为逆时针
单位deg，默认旋转中心是元素的中心点
transform-origin: x y; (转换中心点)可以加方位名词
top/lett/right/bottom
scale缩放(可设置转换中心，默认中心缩放，不影响其他的盒子)
transform: scale(x,y);
transform: scale(1,1);宽，高放大一倍
transform: scale(2,2);宽高放大两倍
transform: scale(0.5,0.5);缩小
