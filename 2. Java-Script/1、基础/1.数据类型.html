常见的数据类型

数字型
八进制程序里面数字前面加0表示八进制,012 = 十进制10
十六进制数字的前面加0x表示十六进制,0x12 = 十进制18
Number.MAX_VALUE最大值：1.7976931348 623157e+308
Number_MIN_VALUE最小值：5e-324
isNaN（）判断非数字，并且返回一个值，如果是数字返回 false，如果不是数字则返回 true
字符型
length检测字符串长度（变量名 . length）
+字符串的拼接
布尔型
false参与加法运算当0来看
true参与加法运算当1来看
typeof()检测数据类型
数据类型转换
数字转字符串1. 变量名 . toString( )
2. String(变量名)
转换成数值型1. parseInt(变量名) （整数）
2. parseFloat(变量名) （小数/浮点数）
Number(变量名) （强制转换）
转换成数值型 - * 利用算数运算（隐式转换）
转换布尔型Boolean( )
结果
undefined为赋值，未定义
null空值（算法当中加上任何数都等于它本身）
Infinity无穷大
-Infinity无穷小
NaN代表一个非数值值(Not a number)
