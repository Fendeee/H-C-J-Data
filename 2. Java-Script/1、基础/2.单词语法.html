函数语法用途
alert()浏览器弹出警示框
console.log()浏览器控制台打印输出信息(控制台可看)
console.error()打印红色警示
console.warn()打印黄色警示
console.dir()打印我们返回的元素对象 更好的查看里面的属性和方法
prompt()浏览器弹出输入框，用户可输入
onclick()点击触发
var 变量名 = 赋值;JS关键字，用来声明变量和赋值，声明多个变量用逗号隔开
undefined只声明不赋值，程序的输出结果会是undefined
变量规范( A~Z \ a~z \ 0~9 \ _ \ $ )不能数字开头
驼峰写法：变量名首字母小写，后面单词首字母大写，如：myFirstName
if、else if条件语句
三元表达式条件语句（var num=2; var su = num >1 ? '对':'不对'）
switch(表达式)switch(表达式){ case 值: 执行语句1; break; default: 执行最后的语句;}
switch语句中的值和数据类型必须一致
for循环
continue退出第n个循环
break推出第n个循环后的整个循环
数组创建方式
new Array（数组长度）;var arr = new Array( );
[ ]var arr = [ ];
添加数组
1.数组名 . push(参数);在数组末尾添加一个或者多个元素，返回值是数组长度
2.数组名 . unshift(参数)在数组的开头添加一个或者多个元素，返回值是数组长度
删除数组
数组名 . pop(参数)删除最后一个数组元素，返回值是数组最后一个元素
数组名 . shift(参数)删除数组中的第一个元素，返回值是数组第一个元素
数组排序
数组名 . reverse( )颠倒数组中元素的顺序，会改变运来的数组返回新数组
数组名 . sort(function (a,b) { return a - b; })对数组元素进行排序（从小到大），会改变原来的数组，返回新的数组
数组名 . sort(function (a,b) { return b - a; })对数组元素进行排序（从大到小），会改变原来的数组，返回新的数组
数组索引方法
indexOf(元素值),[起始位置（可不写）]数组中查找给定元素的索引(数组地址)顺序，如果存在，返回索引号；不存在，返回-1
lastIndexOf(元素值)数组中查找给定元素的索引(数组地址)倒序，如果存在，返回索引号；不存在，返回-1
数组转换字符串
toString( )把数组转换成字符串，逗号分隔每一项
数组 . join('分隔符')用于把数组中的所有元素转换成一个字符串
根据位置返回字符
charAt(索引号)返回指定位置的字符
charCodeAt(索引号)获取指定位置字符ASCll码，用法：判断用户按下了哪个键
str[索引号]获取指定位置处字符
拼接以及截取字符串
concat( )链接字符串('字符串1','字符串2')
substring(start, length)('截取的起止位置'，'截取几个字符')只会替换第一个字符
替换字符，转换为数组
replace( , )('被替换的字符','被替换的字符')
toUpperCase( )转换大写
toLowerCase( )转换小写
字符串 . split('分隔符')字符串转换成数组
检测是否为数组数组名 instanceof Array （可作为运算符）
检测是否为数组Array isArray(数组) （H5新增方法，ie9版本以上才行）
函数
function --声明函数function 函数名 ( ) { 函数体 }
var 变量名 = function ( ) { }变量声明函数
函数名 + ( ); --调用函数
arguments储存了传递过来的所有实参
立即执行函数 不需要调用，立马就能执行
写法 (function() {})()   或者
(function() {} ());
第二个括号看作调用函数，可以传递参数
作用域变量名在某个范围内起作用和效果
全局作用域整个script标签 或是一个单独的js文件
局部作用域（函数作用域）在函数内部就是局部作用域，这个代码名字只在函数内部起效果和作用
全局变量作用域全局作用域下的变量
局部变量作用域在局部变量下的变量
