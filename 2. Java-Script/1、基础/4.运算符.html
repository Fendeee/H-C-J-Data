运算符优先级

优先级运算符顺序
1小括号( )
2一元运算符++ 、 -- 、 !
3算数运算符先 * 、 / 、 % 后 + 、 -
4比较运算符> 、 >= 、 < 、 <=
5相等运算符== 、 != 、 === 、 !==
6逻辑运算符先 && 、 后 ||
7赋值运算符=
8逗号运算符,

运算符

运算符描述实例
1.算数运算符
+加法
-减法
*乘法
/除法
%取余数（去模）返回除法的余数，0代表能被整除 9 % 2 = 1
+=、-=加、减一个数，在赋值var num = 5; num += 3+2; //num = 10
*=、/=、%=乘、除、取余，在赋值var num = 2; num *= 5; //num=10
++递增++sum就是自加1
sum++(先返回原值再自加1)
--递减--sum就是自减1
sum--(先返回原值再自减1)
2.比较运算符比较完后返回一个布尔值(true/false)
<小于
>大于
<=小于等于（小于或者等于）
>=大于等于（大于或者等于）
==判等（会转型）判断是否相等
！=不等于
===全等 要求值和数据类型全部一致
！==全部否定
3.逻辑运算符
&&逻辑与
||逻辑或
！逻辑非
逻辑与短路运算如果表达式1为真，返回表达式2。则返回表达式10 '' null undefined NaN为真，其余为假
逻辑或短路运算如果表达式1为真，返回表达式1，则返回表达式2
