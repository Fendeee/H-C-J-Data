传统注册方式

  • 利用on开头的事件onclick
  • <button onclick="alter('hi~')"></button>
  • btn.onclick = function( ) { }
  • 特点：注册事件的唯一性，
    同一个元素同一个时间只能设置一个处理函数，
    最后注册的处理函数将会覆盖前面的注册处理函数

改变元素文本内容
元素/变量.innerText = ' ';可以不用添加事件直接使用，一刷新就会改变从起始位置到终止位置的内容，但它去除html标签，同时空格和换行也会去掉
标签/变量 .innerHTML可以识别html标签起始位置到终止位置的全部内容,包括html标签,同时保留空格和换行
两者均可获取标签内容console.log(标签.innerText\HTML);
